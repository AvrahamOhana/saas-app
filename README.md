# SaaS App Template
SaaS App Template is a template for a SaaS application, built with React, Vite, and Material-UI.

## Features
- React
- Vite
- Material-UI
- React-Router-Dom

## How to use

Download the example [or clone the whole project]

```bash
git clone https://github.com/AvrahamOhana/saas-app.git
cd saas-app
```

Install it and run:

```bash
npm install
npm run dev
```
